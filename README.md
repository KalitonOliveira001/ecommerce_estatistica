# ecommerce_estatistica

Este projeto faz parte do curso de análise de dados.

## Objetivo
Analisar dados de e-commerce para identificar estatísticas importantes como:
- Produtos mais vendidos
- Categorias com maior receita
- Perfil dos clientes

## Tecnologias usadas
- Python
- Pandas
- Google Colab
